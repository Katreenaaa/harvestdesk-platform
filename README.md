## Overview

HarvestDesk is a **frontend prototype** that explores how a digital dashboard could support farmers with access to agricultural information, sustainability concepts, and clean-energy awareness.

The project focuses on **responsive design, structured navigation, and accessible UI**, rather than backend services or real-time data integration.
